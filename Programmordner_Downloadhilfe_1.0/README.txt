INSTALLATION:
--------------------------------------------------------------------------------------------------------------------------------------------

DISCLAIMER
Die Installation von Python schwächt die Sicherheit des Computers. Ein Angreifer kann, wenn er die Pythoninstallation findet Pythonscripts ausführen.
Aufgrund dieses Risikos sollten Sie das Kästchen "Add Python to PATH" in der Installation freilassen, wenn Sie nicht selber programmieren

--------------------------------------------------------------------------------------------------------------------------------------------
1. Damit das Programm funktioniert, muss auf Ihrem PC "Python 3.9" installiert werden (https://www.python.org/downloads/release/python-390/)
	Dazu:
	-> Runterscrollen bis zu "Files"
	-> Je nachdem ob Ihr PC auf einer 32-Bit Architektur oder einer 64-Bit Architektur basiert den Windows x86/x86-64 executable installer runterladen
	-> Den Installer öffnen und auf "Install Now" klicken, untere Kästchen ignorieren
	-> Im nächsten Fenster sollten Sie "Setup was successful" lesen. Sollte das nicht sein, hilft www.google.de in Kombination mit etwaiigen Fehlercodes, die im Fenster zu sehen sind

2. Es werden zusätzliche Pakete benötigt. Nach der Installation von Python müssen Sie eine Shell in Windows öffnen.
	Dazu:
	-> Windows-Taste drücken
	-> "cmd" in das Suchfeld eingeben
	-> "Als Administrator ausführen" auswählen
	-> "py -3 -m pip install BeautifulSoup4" in die geöffnete Shell eingeben
	-> "py -3 -m pip install requests" in die Shell eingeben, wenn der vorherige Befehl fertig ist

3. Ziehen Sie jetzt den Programmordner in den gewünschten Ordner, wo Sie die heruntergeladenen Bücher speichern wollen. Sollten Sie mehrere Ordner haben, in die Sie EBooks speichern, ziehen Sie den Programmordner in den höchstgelegenen Ordner.

4. Lesen Sie die Benutzungshinweise aufmerksam durch.
